This is a voice assistant project made using python
The first version is published
Features:
Voice input accepted(thanks geeksforgeeks https://www.geeksforgeeks.org/python/how-to-make-a-python-program-wait/)
Assistant that speaks using gemini AI
Next steps:
Publish a backend using flask(V2)
Publish a frontend using react native(V3)

To clone this repository :
Get a gemini api key and add it to .env file in AI_KEY variable
Download necesseary python libraries(Note some libraries dont work with specific versions i recommend you use python 3.12.0)
You are good to go
